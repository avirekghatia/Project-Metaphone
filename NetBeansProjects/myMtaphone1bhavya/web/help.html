How to use




This section provides a general walkthrough of the system from initiation through exit. The logical arrangement of the information shall enable the functional personnel to understand the sequence and flow of the system.
Opening the application
Desktop Application
Run the application from the desktop to open it. On opening an interface to interact with the application is seen. It avails a user to do the following operations: Input: A user should input the lyrics to be searched in the space against Enter the lyrics. Approximation: This field allows the user to set an approximation percentage within a range according to the discretion of the user itself. Results: This field displays the results as songs corresponding to a search.
Online Application
Run the application on your browser. For now, the application does not have an Internet address and therefore it has to be run from the IDE. Input: A user should input the lyrics to be searched in the space against Enter the lyrics. Approximation: This field allows the user to set an approximation percentage within a range according to the discretion of the user itself. Results: This interface has a different page for the display of the result.
